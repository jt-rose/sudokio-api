Sudokio API

Welcome to the Sudokio API!

To get the full info on a puzzle, just enter the puzzle after "api/" with an 81 character string. The character 0 can be used to represent unsolved cells, and all numbers should be listed from the topleft of the puzzle to the bottomright (moving right to left across the row before moving on to the next row)


To get more specific info, you can submit the puzzle after the following url's:

  • "/solve-next/" - provide next step in puzzle
  • "/solve-all/" - show all steps until puzzle is completed
  • "/strategy-next/" - show next strategy suggested
  • "/strategy-all/" - show all required strategies found by the solver
